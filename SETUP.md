
# Icon Setup

To complete the extension setup, you need to convert the SVG icon to PNG format:

1. Open icons/icon.svg in a browser or vector graphics editor
2. Export/save as PNG in the following sizes:
   - 16x16 pixels -> icons/icon-16.png
   - 32x32 pixels -> icons/icon-32.png  
   - 48x48 pixels -> icons/icon-48.png
   - 128x128 pixels -> icons/icon-128.png

Or use an online SVG to PNG converter like:
- https://convertio.co/svg-png/
- https://cloudconvert.com/svg-to-png

## Testing the Extension

1. Open Chrome and go to chrome://extensions/
2. Enable 'Developer mode' in the top right
3. Click 'Load unpacked' and select this directory
4. Visit any website and click the extension icon to test!
